02 — What UniMate Does

One platform. Four core features.

AI-powered course selection that actually understands UNSW.

AI Course Advisor

Analyzes your WAM, progression requirements, workload tolerance, and career goals. Recommends optimal course combinations per semester — fully personalized.

AI: recommendation engine

Timetable Optimizer

Full course catalog from myUNSW JSON API — every course, every section, every time slot. Constraint solver: no 8am, Fridays off, minimize gaps. Computed, not drag-and-drop.

AI: constraint optimization

Vacancy Sniper

Real-time monitoring of enrollment data via myUNSW API. The second a full class opens a spot, you get notified. React in seconds, not hours of refreshing.

Course Intelligence

AI-aggregated reviews, difficulty scores, workload data, and grade distributions. Make informed decisions — not guesses based on one friend's opinion.

AI: sentiment analysis
